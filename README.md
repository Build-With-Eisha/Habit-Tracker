# 📊 Habit Tracker

A simple and visually engaging **Habit Tracker Dashboard** designed to help you stay consistent, track progress, and build powerful daily habits.

---

## 🚀 Features

- 📈 **Progress Tracking** – Shows overall completion %
- 🎯 **Current Streak** – Tracks your weekly consistency
- 📊 **Evaluation Board**
  - Tasks completed per day
  - Consistency per task
- 📅 **Weekly Habit Table** (Monday–Sunday)
- 📂 **Weekly Submission System** – Stores each week's data in a separate sheet
- 📝 **Notes System** – Add notes for better analysis
- ⚡ **Automated Calculations** (via VBA & formulas)
- 🛠️ **Flexible Editing** – Add or modify rows as needed
- 💡 **Motivational Section**

---

## 🛠️ How to Use

1. Open the habit tracker file (**Excel recommended**)

2. Each task has a row and each day has a column

3. By default:
   - **`O` = Not Completed ❌**
   - All cells are already filled with **O**

4. When you complete a task:
   - Replace **`O` → `P`**
   - **`P` = Completed ✅**

5. Update this daily

6. At the end of the week:
   - **Submit your weekly tracker**
   - Add **notes before submitting**
   - Your data will automatically be saved in **weekly tracker** sheet
   - Each submission is stored separately for tracking long-term progress

7. Why notes are important:
   - Helps you understand **where you are lagging**
   - Makes your progress **more meaningful**
   - Helps in **self-analysis and improvement**

8. You can:
   - Add more rows
   - Modify existing rows as per your needs

9. The tracker will automatically:
   - Calculate **progress %**
   - Update your **streak**
   - Show **consistency graphs**

---

## ➕ How to Add More Rows

- You can add rows in the table 
- You can rename or modify existing rows

### Tip:
To add a row safely:
- Insert a row **between two consecutive rows**

This will **not affect formulas** if done correctly.

---

## ⚠️ Important Instructions (Must Read)

- This tracker contains **VBA code/macros** to automate tasks and calculations  
- Kindly **download the file before using it**  
- Some features may not work properly in **online preview mode**

- ❗ Do **NOT delete random cells**
- Some cells contain **formulas (in white font)** — they may look empty but are important

- These formulas help:
  - Calculate progress %
  - Track streak
  - Generate graphs & consistency data

- ⚠️ If you delete or overwrite them:
  - The tracker may stop working properly

- ✔️ Only edit:
  - Tracking cells (where **`O`** is written)
  - Row names
  - Safely added rows

## 📌 Rules to Remember

- Replace only **`O → P`**

- Update daily (takes less than 1 minute)
- Submit your tracker at the **end of each week**
- **Always add notes before submission**
- Do not modify hidden formula cells or important structure
